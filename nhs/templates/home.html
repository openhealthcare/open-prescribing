Open Prescriptions

Every month, the NHS Information Centre publishes an open dataset of the UK's GP prescribing activity. Roughly ten million records each month. Which is brilliant for openness, transparency and enabling research, but not particularly friendly.

Open Prescribing offers a free public API for anyone to build visualisations and answer questions about GP prescribing habits, along with sample applications that make use of the data in interesting ways.

What's this all about then?

So it turns out that the NHS open sources all of it's GP prescribing data.

Which is clearly awesone.

Here, we turn those CSV's into several things:

  • An API that you can call to do useful things with the data
  • A whole bunch of visualisations that are somewhat interesting (Provided you have some intellectual curiosity).

Some examples:

  • Aggregate UK statin prescriptions
  • Aggregate UK statin prescriptions with linked patent data
  • Practice level statin prescribing with linked patent data
  • NICE Recommendations with linked prescribing data
  • Where do we get postcode lottery prescriptions
